# Loan Qualifier Application

This is a python command-line application interface application that evaluates a user's xxx loan-to-value, debt, income and informs them if they qualify for a loan. It also allows for them to save the qualifying loans as a new CSV file.

---

## Technologies

This project leverages python 3.7 with the following packages:

* [fire](https://github.com/google/python-fire) - For the command line interface, help page, and entrypoint.

* [questionary](https://github.com/tmbo/questionary) - For interactive user prompts and dialogs

---

## Installation Guide
Before running the application first install the following dependencies:

pip install fire
pipe install questionary

---

## Usage

To use the loan qualifier application simply clone the repository and run the **app.py** with:

```python
python app.py
```

Upon launching the loan qualifier application you will be greeted with the following prompts.
